# Sistema Gerenciador de Produtos
Este projeto consiste em um sistema web desenvolvido em React para gerenciamento de produtos.
A aplicação permite cadastrar, editar, excluir e visualizar produtos, além de oferecer funcionalidades de filtragem e relatórios com gráficos.
O sistema foi desenvolvido com foco na organização de dados e na criação de uma interface moderna e funcional, simulando um ambiente real de gerenciamento.

# Tecnologias Utilizadas
* React (Hooks: useState, useEffect)
* JavaScript
* CSS3
* Recharts para gerar gráficos apertir dos dados.
* Lucide React, icones.
* LocalStorage 

# Estrutura do Sistema
O sistema é dividido em módulos principais:
* **Dashboard** → visão geral dos dados
* **Produtos** → gerenciamento completo dos produtos
* **Relatórios** → gráficos e análise de dados
* **Configurações** → personalização do tema (modo escuro)
* **Sobre** → informações do sistema

# Indicadores (KPIs)
O sistema apresenta métricas importantes como:
**Total de produtos cadastrados**
**Quantidade total em estoque**
**Valor total do estoque**

# Objetivo
O objetivo do projeto é aplicar conceitos de desenvolvimento front-end utilizando React, além de praticar:
**Manipulação de estado
  Componentização
  Organização de código
  Interação com armazenamento local
  Visualização de dados com gráficos**

**Considerações Finais**
Este projeto representa a aplicação prática dos conhecimentos adquiridos ao longo do desenvolvimento em React e no estudo de tecnologias voltadas ao front-end. Durante a construção do sistema, foi possível exercitar conceitos importantes como componentização, gerenciamento de estado, manipulação de dados e criação de interfaces interativas.
Inicialmente, a proposta deste trabalho não era a criação de um sistema de gerenciamento de produtos, mas sim o desenvolvimento de uma plataforma de vendas online. No entanto, devido à limitação de tempo para a execução do projeto, foi necessário realizar uma adaptação do escopo, resultando na implementação de um sistema de gerenciamento mais simples, porém funcional e bem estruturado.
Mesmo com essa mudança de direção, o projeto manteve seu objetivo principal de consolidar o aprendizado e simular uma aplicação próxima da realidade. Dessa forma, foi possível construir uma base sólida de funcionalidades, incluindo cadastro de produtos, relatórios e visualização de dados, o que contribui significativamente para a evolução técnica e acadêmica do desenvolvimento.
Por fim, o sistema demonstra a capacidade de adaptação durante o processo de desenvolvimento e reforça a importância do planejamento e da organização em projetos de software. Apesar das limitações, o resultado final atende aos objetivos propostos dentro do tempo disponível e serve como base para futuras melhorias e evolução para um sistema mais completo de comércio eletrônico.
